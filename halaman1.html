TOPIK 7

Pada topik ke tujuh ini adalah untuk pembahasan tag tabel dari HTML. Tabel sendiri berfungsi untuk menampilkan suatu data dari website. Tabel terdiri dari baris dan kolom, lalu perpotongan antara baris dan kolom disebut sebagai cell. Dalam cell inilah data website akan dimasukkan. Struktur tag tabel terdiri dari (table), lalu table row (tr) dan table data (td). Untuk mengetahui bagaimana susunan dari tabel adalah pada halaman Pemrograman Web 3
